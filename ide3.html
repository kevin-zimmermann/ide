Teletype pour Atom.

De grandes choses se produisent lorsque les développeurs travaillent ensemble, de l'enseignement et du partage des connaissances à la création de meilleurs logiciels. Grâce au Télétype pour Atom, il est aussi facile de collaborer sur du code que de coder seul, directement depuis votre éditeur.

Partagez votre espace de travail et modifiez le code ensemble en temps réel. Pour commencer à collaborer, ouvrez Teletype in Atom et installez le paquet.

GitHub pour Atom

Un éditeur de texte est au cœur de la boîte à outils d'un développeur, mais il ne fonctionne généralement pas seul. Travaillez avec Git et GitHub directement depuis Atom avec le package GitHub.

Créez de nouvelles branches, mettez en scène et validez, poussez et tirez, résolvez les conflits de fusion, visualisez les demandes de retrait et plus encore, tout cela depuis votre éditeur. Le pack GitHub est déjà fourni avec Atom, vous êtes donc prêt à partir !

Tout ce que vous attendez.

  • Multipltaformes : Atom fonctionne sur les systèmes d'exploitation OS X, Windows ou Linux.
  • Gestionnaire de paquets intégré : Atom propose d'installer de nouveaux paquets ou de créer le sien.
  • Autocomplétion intelligente : Atom vous permet d'écrire du code plus rapidement avec une auto-complétion intelligente et souple.
  • Navigateur de système de fichiers : Atom permet de facilement naviguer et ouvrir un fichier unique, un projet entier, ou des projets multiples dans une seule fenêtre.
  • Fenêtres multiples : Divisez votre interface Atom en plusieurs panneaux afin de comparer et de modifier le code.
  • Rechercher et remplacer : Trouver et remplacer du texte dans un fichier ou dans tous vos projets.
